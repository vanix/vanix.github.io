# 課程內容

## 1. Python 文字輸出功能（Print）
- **功能介紹**: Print 函數用於將資料顯示在畫面上。
- **程式碼範例**:
  ```python
  print("Hello World!")
  print("Hello Covid-19")
  ```
  - **執行結果**:
    ```
    Hello World!
    Hello Covid-19
    ```

## 2. 顯示在同一行的資料
- **方法一：使用逗號分隔**:
  - **程式碼範例**:
    ```python
    print("Hello", "World!")
    ```
    - **執行結果**:
      ```
      Hello World!
      ```

- **方法二：修改結尾符號**:
  - **程式碼範例**:
    ```python
    print("Hello", end="...")
    print("World!")
    ```
    - **執行結果**:
      ```
      Hello...World!
      ```

## 3. 取得用戶輸入（Input）
- **功能介紹**: 使用 input 函數來取得使用者輸入的資料。
- **程式碼範例**:
  ```python
  name = input("請輸入您的名字: ")
  print("哈囉", name)
  ```
  - **執行結果**:
    ```
    請輸入您的名字: 小明
    哈囉 小明
    ```

## 4. 設定變數
- **變數定義**: 變數是一個用來儲存資料的容器。
- **程式碼範例**:
  ```python
  age = input("請輸入您的年齡: ")
  print("您的年齡是", age)
  ```
  - **執行結果**:
    ```
    請輸入您的年齡: 25
    您的年齡是 25
    ```

## 5. 字串合併
- **功能介紹**: 可以通過加號 (`+`) 將字串合併。
- **程式碼範例**:
  ```python
  name = input("請輸入您的名字: ")
  greeting = "哈囉, " + name + "!"
  print(greeting)
  ```
  - **執行結果**:
    ```
    請輸入您的名字: 小華
    哈囉, 小華!
    ```

## 6. 課程總結
- **重點回顧**: 
  - Print 用於輸出資料。
  - Input 用於取得用戶資料。
  - 變數用於儲存資料。
  - 字串合併來組成完整的訊息。

- **練習**: 學員可以嘗試使用 Print 和 Input 函數進行更多的練習。
