adb problem

ref: [Remote debug Android by adb tool](http://jackytw.blogspot.com/2010/05/remote-debug-android-by-adb-tool.html)

$ mv /dev/android\_adb /dev/android\_adb\_bk

$ kill adbd process

set the capture volume to 0

$ alsa\_amixer set 'Analog' 0

其他指令

$ alsa\_amixer get 'Headset'

$ alsa\_amixer get 'Analog'

make boot.scr

ref: [BeagleEditBootscr](https://wiki.ubuntu.com/ARM/BeagleEditBootscr)

$ dd if=boot.scr of=boot.script bs=1 skip=72

$ mkimage -A arm -T script -C none -n "Ubuntu boot script" -d boot.script boot.scr
