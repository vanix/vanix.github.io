你知道嗎？現在在Mac上也可以順利地運行PC遊戲，效能表現也相當不錯！今天，我會介紹如何在Mac上安裝PC遊戲，並以《魔獸兵團》為例，提供詳細的操作步驟，讓你事半功倍地開始遊玩。

首先，我嘗試了幾種不同的軟體來安裝Battlenet。Crossover是可以的，但需付費；Whiskey則無法運行；而Porting Kit宣稱可以，但我沒有測試到一半就沒測了。最後，我發現Wine Skin能夠成功運行並且是免費的。所以接下來的教學將圍繞Wine Skin展開。

### 懶人包

1. \*\*安裝Homebrew和Wine Skin\*\*

- 打開Terminal，執行以下指令來安裝Homebrew：

```

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

```

- 安裝完成後，複製並在終端機中執行畫面提示的三行指令。

- 接著，輸入以下指令來安裝Wine Skin：

```

brew install --cask --no-quarantine gcenx/wine/wineskin

```

2. \*\*配置Wineskin\*\*

- 開啟Wineskin，並安裝Rosetta。

- 然後，安裝Engine和Wrapper。

- 建立一個新的容器。

3. \*\*下載和安裝Battlenet\*\*

- 下載battle-net.exe文件。

- 在Wineskin容器中，點選「Install Software」，選擇下載好的battle-net.exe並執行。

- 設定語言為英文，開始安裝Battlenet。

- 如果顯示安裝失敗，重試一次通常能成功。

4. \*\*安裝《魔獸軍團》\*\*

- 在Battlenet中，直接到魔獸軍團頁面按下「開始安裝」。

- 如果畫面卡住，等待更新完成後，關掉《魔獸兵團》再啟動一次。

5. \*\*配置容器設定\*\*

- 右鍵點選容器，顯示套件內容。

- 開啟套件內的Wineskin，設定執行檔路徑為：

```

C:\Program files(x86)\Battle.net\Battle.net Launcher.exe

```

- 勾選「Direct 3D to Metal」選項。

- 關閉設定視窗，回到容器的資料夾。

6. \*\*啟動遊戲\*\*

- 點擊容器，即可直接開啟Battlenet程式，進而開始遊玩《魔獸兵團》。

這篇指南旨在協助Mac用戶能順利安裝及運行PC遊戲。不僅限於《魔獸兵團》，相同的步驟也適用於其他更多的PC遊戲。希望對大家有所幫助，如果有成功體驗其他遊戲，歡迎分享你的經驗！祝各位遊戲玩得愉快！
